# Todo List

此作品能讓使用者新增代辦事項及刪除完成事項，主要練習React,也同時熟悉Chakra UI的使用方式，並使用Local Storage將資料存在本地空間。

Link: https://todo-list-zeta-jade.vercel.app/

![Imgur](https://i.imgur.com/O4g4jeN.png)

# 功能

- 新增事項
- 移除事項
- 使用Local Storage
- 光暗模式切換

# 使用技術

Client: React, Chakra UI

# 安裝

要安裝此應用程式，請按照以下步驟操作：

開啟終端機，並進入此專案的目錄。

執行以下指令以安裝相關套件：

```
npm i
```

# 執行

要啟動此應用程式，請執行以下指令：

```
npm start
```
